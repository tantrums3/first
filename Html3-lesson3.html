Using Attributes
When writing the document.getElementById tag, there will need to be instructions that follow the period.
The instructions will always pertain to HTML, so therefore you need to be familiar the HTML attributes that will be possibilities to use with the [document.getElementById] tag.
There are several version of HTML ranging from 2.0 to 5 but any version before 5 is deprecated (out-dated) and do not work with well with newer versions of other technology (JavaScript, CSS, browser, etcetera).
So, be aware of what you are working with before you pick a version of HTML to use.

HTML Attributes:
  • innerHTML
  • src
  • style.display
  • style.fontSize

More HTML attributes can be found at https://www.w3schools.com/tags/ref_attributes.asp.
