Hello Visitor!
Welcome to the tutorials! Much of the lessons are based on content available at the w3schools website.
The tutorials are created with very little css because there is intent on keeping the lessons simple.
The purpose of the tutorials is to make it easy to learn JavaScript.
Before we start, there is something that has to come first.


Let us start by checking if you have JavaScript enabled within the blue box below.

Message appeared, what now?

If you do not have JavaScript enabled then you will be seeing a message in the box below.
If there is a message, then you need to go to your browser's menu, find the settings, and within the settings find where JavaScript is enabled/disabled then change that setting.

No message, what does that mean?

If no message appears then you will be able to work with the lessons from the tutorials and the container below will remain empty.
