Making Changes
Previously, the first half of the JS code was explained to show how it creates the id.
Now in this lesson is the review of the part of the JS code that creates the action, through the HTML portion of the getElementById attribute.
In the last lesson, .innerHTML was used to display the date and time.
This time HTML instructs what to change in the display.
The HTML is going to be used to to change an attribute of an element tag and it will be used to change the variant of an attribute.
In the first portion of the sample, you see the HTML element for source being used to swap-out the image when the button is clicked.
In the second portion of the sample, you see the HTML element for Style being used to make the image vanish and return.
The samples just show that any HTML element can be used to make changes, so long the id is created with the document.getElementById part of the attribute.
